- [遇到公司安排你居家办公，这该怎么办？](https://mp.weixin.qq.com/s/QPQqp7F6i7Ndk_Q7yMMMVQ) - 对于很多传统公司和岗位，居家办公仍然是一种禁忌，公司和员工对彼此都充满了不信任感，在一个没有远程基因的公司突然被要求远程办公，很多打工人难免会手足无措，这种情况下，做好必要的保护措施还是有必要的，比如要求书面通知，保留好证据，避免考勤上面的纠纷。信任是远程协作的基石，如果没有信任基础，那么协作无疑是痛苦的，如果真的想远程工作，还是建议去一个有远程基础的团队。
- [周末居家办公猝死算工伤吗？](https://mp.weixin.qq.com/s/AKPLWNSg_yqSkR88IttW9w) - 居家办公的工伤认定问题一直存在争议，尤其是周末在家办公发生意外时。法律界对于工作时间和工作场所的界定尚无统一标准。这篇文章中的案例因为有很清晰的证据证明员工在家处理公司业务，才最终得以认定工伤。根据《工伤保险条例》第十五条第一款第一项的规定，职工“在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡的”，视同工伤。“工作岗位”不同于“工作场所”，更着重于强调职工是否在履行岗位职责、完成工作任务。居家办公突破了常规工作时空限制，但同样可能构成“工作时间”和“工作岗位”。
